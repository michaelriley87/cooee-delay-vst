# Cooee Delay: Simple Delay VST

## Introduction
**Cooee Delay** is a lightweight delay plugin written in C++ using the JUCE framework.  
The project was built as a focused exploration of real-time audio DSP, implementing a classic feedback delay with tone shaping controls inspired by dub and reggae delay units.

The plugin demonstrates the fundamentals of audio plugin development including circular delay buffers, feedback processing, parameter management, and real-time safe DSP execution.

## Features
- **Delay Time Control:** 1 ms – 2000 ms delay range
- **Feedback Control:** Adjustable repeat intensity
- **Wet/Dry Mix:** Balance original signal and delay output
- **Low-Cut Filter:** High-pass filter in the feedback path
- **High-Cut Filter:** Low-pass filter in the feedback path
- **Stereo Support:** Independent processing for each channel
- **Real-Time Safe DSP:** No memory allocations during audio processing

## Installation
Cooee Delay can be used either as a **VST3 plugin inside a DAW** or as a **standalone desktop application** generated by JUCE.

1. **Build the Project:** Open the JUCE project in Visual Studio and build the solution.
2. **Locate the Plugin:** After building, the compiled VST3 plugin can be found in the project's build directory.
3. **Install the Plugin:** Copy the `.vst3` file into your system VST3 directory (typically `C:\Program Files\Common Files\VST3` on Windows).
4. **Load in a DAW:** Open your DAW and rescan plugins. The delay should appear as **Cooee Delay**.
5. **Standalone Use:** JUCE also builds a standalone executable which can be launched directly, allowing the delay to run without a DAW for testing or experimentation.

## Technologies Used
- **C++:** Core implementation language
- **JUCE Framework:** Audio plugin architecture and GUI framework
- **JUCE DSP Module:** Filter implementation and DSP utilities
- **VST3:** Plugin format
